- box모델

  width = 넓이 (부모기준!!!)
  min-width = 넓이에 최소값 지정, 지정된 값 이하는 줄어들지 않는다.
  max-width = 넓이에 최대값 지정, 지정된 값 이상은 커지지 않는다.

  height = 높이 (컨텐츠 기준!!!)
  min-height = 최소높이
  max-height = 최대높이

 절대값:px
 상대값:%
 
  margin = 바깥쪽 여백 (*margin은 안보이는 영역이 커진것이지 이동한것이 아니다!!!!)
  margin-top
  margin-right
  margin-bottom
  margin-left

  padding = 안쪽 여백
  padding-top
  padding-right
  padding-bottom
  padding-left

  border = 선의 두께
  border-width
  border-style
  border-color

  margin:0 auto; = auto는 내 부모만큼만 100%로 생각한다!!
  width:auto; = 부모만큼
  height:auto; = 컨텐츠만큼

   * float 보다 먼저 선언된 부분 위로 뜰수 없다!!
